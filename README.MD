# Liiga Widgets

A collection of browser-native web components for displaying Liiga stats and data.

This project is not affiliated with Liiga. The widgets load data from Liiga's public API, which may change or become unavailable at any time.

## Installation

Include the stylesheet and JavaScript bundle from jsDelivr.

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/niilopoutanen/liiga-widgets@v0.1.1/dist/widgets.css">

<script
  type="module"
  src="https://cdn.jsdelivr.net/gh/niilopoutanen/liiga-widgets@v0.1.1/dist/widgets.js">
</script>
```
